html+css tips

  1. 浮动的元素和加了绝对定位的元素可以直接给大小

  2. 子绝对位置 父相对位置 “子绝父相”

  3. 垂直居中，top 50%，再往上走自己高度的一半

  4. 半透明写法 rgba

  5. CSS 属性 border-radius 可以设置元素的外边框圆角

    /鼠标经过显示小手/

    cursor: pointer;

  6. Border : 像素 样式(a b c) 颜色

    1. solid 实线
    2. Dotted 点线
    3. Dashed 虚线
  8. 默认是竖着排列的，要浮动

  9. overflow: hidden;溢出部分隐藏

  10. /让加粗的不变粗/

    font-weight: normal;
